## 适用场景

- **List**: 适合展示轻量级结构化信息，支持索引背景、头部栏、选择框、尾部操作多种组合
- **UserList**: 适合需要快速构建列表页面的场景，组件内置了分页和加载功能
- **UserTable**: 适合需要复杂数据展示和操作的场景，支持更多高级功能
