# Changelog

## [0.1.0] - 2024

### 新增
- 项目初始化，基于 Vant 2.x 实现 Element-UI 兼容组件库
- 实现 65 个 `el-` 前缀组件，覆盖 Basic、Form、Data、Navigation、Feedback 五大分类
- 实现 4 个服务式 API：`$message`、`$notify`、`$loading`、`$alert` / `$confirm` / `$prompt` / `$msgbox`
- 支持完整引入、按需引入和 babel-plugin-import 自动按需引入
- 提供 VuePress 2 文档站和使用示例
